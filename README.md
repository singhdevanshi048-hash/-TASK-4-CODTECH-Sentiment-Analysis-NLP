## 📌 Project Overview

This project performs **Sentiment Analysis** on customer reviews using **Natural Language Processing (NLP)** and **Machine Learning**.

The system classifies user reviews into **Positive** or **Negative** sentiments by applying **TF-IDF Vectorization** and **Logistic Regression**. A professional **Streamlit Dashboard** has also been developed to provide an interactive user experience.

---

## 🎯 Objectives

- Perform sentiment analysis on textual data.
- Apply Natural Language Processing techniques.
- Train a Machine Learning classification model.
- Predict sentiment from user input.
- Build an interactive dashboard for visualization.
